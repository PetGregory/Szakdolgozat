# BeStrong - Szakdolgozat

Angular alapú fitness alkalmazás edzésterv készítéséhez, kalória követéséhez és edzések rögzítéséhez.

## Hosztolás

- **Élő URL**: https://bestrong-74932.web.app

## Telepítés és Futtatás

### 1. Projekt klónozása

```bash
git clone <repository-url>
cd beStrong
```

### 2. Függőségek telepítése

```bash
npm install
```

### 3. Alkalmazás futtatása fejlesztési módban

```bash
npm start
```

Az alkalmazás elérhető lesz a `http://localhost:4200` címen.
